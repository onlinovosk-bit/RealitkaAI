# Orchestrator 2 — Produktové UX & povrch CRM

## Úloha

Koordinuje úlohy okolo **používateľského rozhrania a produktového zážitku** v CRM vrátane spätnej väzby cez analytics a notifikácie v aplikácii.

## Riadi (scope)

| Oblast          | Popis |
| --------------- | ----- |
| **UI**          | stránky, komponenty, stavy načítania, prístupnosť |
| **onboarding**  | prvé kroky, prázdne stavy, guidovanie |
| **CRM UX**      | práca s leadmi/deal pipeline z pohľadu používateľa |
| **analytics**   | meranie použitia v aplikácii (eventy relevantné pre CRM) |
| **notifications** | užívateľské kanály v produkte, preferencie, copy |
